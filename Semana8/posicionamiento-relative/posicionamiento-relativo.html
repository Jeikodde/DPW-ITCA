Posicionamiento Relativo

Este párrafo está contenido en una caja con posicionamiento relativo. Esto significa que se mueve respecto a su posición original. Puedes usar esto para ajustes finos de diseño.

Otro párrafo en la misma caja para mostrar cómo se comporta el contenido cuando el contenedor tiene desplazamiento. Este tipo de posicionamiento no saca al elemento del flujo normal.

El uso de posicionamiento relativo es común cuando necesitas mover un elemento ligeramente sin afectar a los demás elementos de la página.

El posicionamiento relativo en CSS es una técnica que permite mover un elemento en relación con su posición normal en el flujo del documento. Esto se logra utilizando la propiedad position: relative junto con las propiedades de desplazamiento como top, right, bottom y left.
