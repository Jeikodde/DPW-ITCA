Guía de Controles HTML para Formularios

1. Campo de texto ()

Se utiliza para capturar texto como nombres o direcciones.

2. Selector de fecha ()

Permite al usuario seleccionar una fecha.

3. Casilla de verificación ()

Se utiliza para seleccionar múltiples opciones.

4. Botón de opción ()

Se utiliza para seleccionar una sola opción.

5. Área de texto ()

Se utiliza para escribir texto más extenso como comentarios.

6. Botón de envío ()

Permite enviar la información del formulario.

7. Menú desplegable ()

Permite seleccionar una opción de una lista.
